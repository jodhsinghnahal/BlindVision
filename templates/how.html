Welcome to BlindVision App Guide

Usage Instructions

Triple tap or double tap highlighted area to swtich modes

Desktop

Use arrow keys

Mobile

Swipe 4 directions in highlighted area if in draw mode, see below

Swipe-down/up-arrow-key: speech recognition

Swipe-up/down-arrow-key: speak the entered text.

Swipe-right/left-arrow-key: speak the received response from the model.

Swipe-left/right-arrow-key: send the message (image and/or entered text) to the model.

Draw Mode

Drawing mode: Draw characters on the canvas using mouse or tap.

Quickly tap/click: clear the canvas.

Quickly tap/click again: add enetered text.
